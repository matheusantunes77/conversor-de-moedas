<h1>Sobre o projeto</h1>
<p>Esse conversor foi desenvolvido com a linguagem de marcação de texto HTML e de estilização CSS e a linguagem de programação JavaScript.</p>
<h1>Funcionalidades</h1>
<p>Onde tem um imagem GIF para interatividade do usuário e depois o conversor pedindo a moeda que você está usufruindo e para a qual moeda você quer usar e o valor que você quer converter.</p>
<p>Tem as opções de moeda como Real, Euro, Libra , Dinar kwatiano.</p>
